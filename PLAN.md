# 貪食蛇遊戲開發計畫

## 專案概述
使用 Python `pygame` 函式庫開發一款單機貪食蛇遊戲。

## 技術棧
- 語言：Python 3
- 函式庫：`pygame`

## 檔案結構
- `snake_game.py`: 遊戲主程式，包含遊戲迴圈、渲染邏輯與碰撞偵測。
- `requirements.txt`: 專案依賴清單 (pygame)。

## 開發步驟
1.  **環境設定**：安裝 `pygame` (透過 `pip install pygame`)。
2.  **核心邏輯**：
    - 初始化 `pygame` 視窗。
    - 定義蛇的數據結構（身體座標陣列）。
    - 實作鍵盤監聽事件以改變移動方向。
    - 實作遊戲迴圈 (Game Loop)。
    - 實作食物生成機制。
3.  **碰撞與結算**：
    - 檢測是否吃到食物。
    - 檢測是否撞牆或撞到自己。
    - 實作遊戲結束與重新開始邏輯。
4.  **測試**：執行並進行遊玩驗證。
