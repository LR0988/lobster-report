主要技術棧：
- 後端：Python 3.11 + FastAPI + PostgreSQL + Redis
- 前端：React 18 + TypeScript + Tailwind CSS
- 部署：Docker + Docker Compose
- 爬蟲：Scrapy / Playwright
- AI：OpenAI API / Local LLM
