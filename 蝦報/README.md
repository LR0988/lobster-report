# 蝦報 (Shrimp Report) - 整體架構框架

## 專案概述
**蝦報** 是一個即時資訊聚合與分析系統，專注於新聞、財報、世界局勢與網路論壇討論的快速分析，並透過網頁介面提供即時資訊掌握。

## 技術棧

### 後端 (Backend)
- **語言：** Python 3.11+
- **框架：** FastAPI (高效能 API)
- **資料庫：** PostgreSQL (結構化資料) + Redis (快取)
- **任務排程：** Celery + Redis
- **爬蟲框架：** Scrapy / Playwright
- **AI 整合：** OpenAI API / Local LLM

### 前端 (Frontend)
- **框架：** React 18 + TypeScript
- **UI 庫：** Tailwind CSS + Shadcn UI
- **圖表：** Chart.js / Recharts
- **狀態管理：** Zustand
- **即時更新：** WebSocket / Server-Sent Events

### 部署 (Deployment)
- **容器化：** Docker + Docker Compose
- **雲端：** AWS / GCP / Vercel
- **CI/CD：** GitHub Actions

## 資料來源層 (Data Sources)

### 1. 新聞來源
- **台灣新聞：** 聯合新聞網、中央社、自由時報
- **國際新聞：** Reuters、BBC、CNN
- **財經新聞：** 經濟日報、工商時報

### 2. 財報來源
- **台灣：** 公開資訊觀測站 (TSE)
- **美國：** SEC EDGAR
- **中國：** 上海/深圳證券交易所

### 3. 論壇來源
- **台灣：** PTT、Dcard
- **國際：** Reddit、Twitter (X)
- **中國：** 微博、知乎

### 4. 世界局勢
- **國際組織：** UN、WHO、WTO
- **地緣政治：** 戰爭新聞、外交動態
- **經濟指標：** 各國 GDP、通膨數據

---

## 處理層 (Processing Layer)

### 資料清洗與儲存
- 去除重複內容
- 標準化格式
- 儲存至資料庫

### AI 分析模組
- **情緒分析：** 新聞與論壇情緒趨勢
- **關鍵字提取：** 重要議題識別
- **關聯分析：** 跨來源資訊關聯
- **摘要生成：** 自動生成報告摘要

### 趨勢計算
- **熱度指標：** 討論量、搜尋量
- **情緒指數：** 正負面情緒比例
- **關聯圖譜：** 議題關聯網絡

### 警報系統
- **關鍵事件：** 重大新聞觸發警報
- **異常波動：** 財報或股價異常
- **趨勢轉變：** 情緒或熱度突變

---

## 展示層 (Presentation Layer)

### 網頁介面
- **首頁：** 總覽儀表板
- **新聞頁面：** 即時新聞列表與分析
- **財報頁面：** 公司財務報告
- **論壇頁面：** 討論趨勢與熱門話題
- **世界局勢頁面：** 國際動態與地緣政治

### 功能特色
- **即時更新：** 自動刷新最新資訊
- **自訂儀表板：** 用戶可自訂顯示內容
- **搜尋功能：** 關鍵字搜尋與過濾
- **匯出功能：** PDF/Excel 報告下載

---

## 資料庫設計

### 主要資料表
1. **news_articles** - 新聞文章
2. **financial_reports** - 財務報告
3. **forum_posts** - 論壇貼文
4. **world_events** - 世界局勢事件
5. **analysis_results** - AI 分析結果
6. **users** - 用戶資料
7. **alerts** - 警報記錄

---

## API 設計

### 主要端點
```
GET    /api/news           - 取得新聞列表
GET    /api/news/:id       - 取得單則新聞
POST   /api/news/search    - 搜尋新聞

GET    /api/financial      - 取得財報列表
GET    /api/financial/:id  - 取得單筆財報

GET    /api/forum          - 取得論壇貼文
POST   /api/forum/search   - 搜尋論壇

GET    /api/world          - 取得世界局勢
POST   /api/alerts         - 建立警報
GET    /api/dashboard      - 取得儀表板資料
```

## 開發環境設定

### 前置需求
- Python 3.11+
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL
- Redis

### 快速開始
```bash
# 克隆專案
git clone <repository-url>
cd shrimp-report

# 啟動後端
cd backend
pip install -r requirements.txt
python main.py

# 啟動前端
cd ../frontend
npm install
npm run dev

# 啟動資料庫
docker-compose up -d
```
